Este proyecto es para crear código en C y poder ir construyendo las funciones que se necesitan
para computar los valores de los puntos de las señales que es necesario inyectar desde el DAC.
Simplemente es código de prueba para ir creando el esqueleto del código que luego va a ser necesario 
implementar en el MCU.

Cuando se programe el equipo de CEMITEC será necesario utilizar las APIs junto con estas funciones 
para preparar el equipo para realizar la técnica deseada.
